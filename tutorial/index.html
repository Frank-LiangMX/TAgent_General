什么是 TAgent

两种模式，一种生产力

通用模式覆盖日常 AI 协作的所有场景，TA 模式深度集成游戏技术美术工作流，两者独立运行，记忆 / 工具 / 会话完全隔离。

通用 Agent 模式

基于 Claude Agent SDK，集成 MCP 工具调用、Skills 技能包、SubAgent 委派、记忆与联网能力，覆盖日常 AI 协作全场景。

TA 领域模式

54 个游戏 TA 工具 + SQLite 资产库 + 审核流水线 + Blender / UE5 集成，把领域 Know-how 直接灌给 Agent。

飞书远程协同

手机端群聊 / 私聊直接派任务，Agent 在 PC 端处理后回传结果。通勤路上也能推进 PR 和深度研究。

快速开始

5 分钟完成安装与配置

下载 → 安装依赖 → 配置渠道 → 开始使用，跟着四步走即可。

1

下载并安装 TAgent

从 GitHub Releases 页面下载最新安装包。Windows 用户推荐安装到 C 盘，对 Agent 执行更友好。

github.com/Frank-LiangMX/TAgent_General/releases
2

安装 Node JS 和 Git

首次启动后按引导流程默认完成即可。Windows 用户务必安装到 C 盘，并勾选安装流程中的 Add to Path 选项。

提示：Node JS ≥ 18，Git ≥ 2.30 即可，无需额外配置。
3

配置 AI 渠道

设置 → 渠道 → 添加渠道。Agent 模式要求渠道支持 Anthropic 协议，填写 BaseUrl 和 API KEY 后点 测试链接 验证。

推荐：DeepSeek、SiliconFlow、官方 Anthropic、月之暗面等任选其一。
4

启用记忆和联网（可选）

MemOS 提供每月免费 5 万次记忆存储 / 2 万次调用；Tavily 提供每月 1000 次免费联网。设置 → 工具 完成配置后，在 Chat 模式右上角打开对应开关。

核心概念

用好 Agent 之前，先理解这四个概念

这四个概念构成了 TAgent 的能力底座，理解后可以自己组合出无限工作流。

Skills

技能包

渐进式披露的技能包，把一类工作的处理流程和判断标准结构化地交给 Agent。写一次，无限复用。

MCP

模型上下文协议

把外部服务（Gmail、飞书、GitHub）包装成 Agent 可调用的标准工具，自然语言对话即可完成安装。

Workspace

工作区

按工作类型拆分独立配置。处理 GitHub PR 一个区，处理 Gmail 另一个区，配置互不干扰。

SubAgent

子代理

复杂任务委派给专家子代理（探索 / 调研 / 审查），主 Agent 调度并展示全过程。

不在电脑前，也能用 TAgent

通过飞书远程连接你的 TAgent Agent。手机端群聊或私聊直接派任务，处理结果自动回传。典型场景：通勤做深度研究、远程处理 PR、紧急修复 Bug。
