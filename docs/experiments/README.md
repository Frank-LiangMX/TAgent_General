# docs/experiments

> **试错记录**：方案探索、PoC 验证、技术调研中产出的非最终结论性文档。

## 命名规范

```
experiments/{backend,frontend,infra,design}/YYYY-MM-DD-<topic>.md
```

例：

- `experiments/frontend/2026-06-XX-mermaid-rendering.md`
- `experiments/infra/2026-06-XX-electron-bundler-comparison.md`

## 与 `archive/` 的区别

| | experiments/ | archive/ |
|---|---|---|
| 用途 | 探索中、可能复活 | 已废弃、仅追溯 |
| 状态 | 仍可参考 | 不再演进 |
